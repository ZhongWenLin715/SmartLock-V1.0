BOMS
MCU:STM32F103ZET6指纹识别：AS608
短信：GPRS A6
wifi模块：esp8266-12F
0.96寸OLED
3V减速电机
蜂鸣器
LED


实现功能
1.指纹识别解锁，可保存300个，支持单个或全部清除指纹
2.密码解锁，密码8位数字，中间4位为有效密码，重设密码时必须设为8位，解锁时只要保证中间4位正确
3.app解锁，烟雾模块读取数值，AD采样，可修改为电量检测。
拥有一键SOS
特殊指纹解锁时会发短信提示
密码输错5次会发短信提示
开门密码和root密码相同时会设置默认为12345678
密码和指纹均支持掉电保存
